# Directories

	src     - Counter implementation
	sim     - Testbench implementation
	model   - python script to generate the reference file for test bench.
	project - tcl script to create the vivado project.

# Reference values

	The script model/refgen.py generates the file with reference values for test bench.

	refgen.py <count_start> <output_file>

	<count_start> is the fixed number from which the counter starts.
	<output_file> is the name of the file created by this script with the reference values.

	The file mode/ref.txt was generated by the command:

	refgen.py 1000 ref.txt
